# Travel Booking System
Приложение за запазване на почивка, създадено с **Spring Boot**.

## Функции
-  CRUD
-  Book best trip(based on price and destination) with **Facade Design Pattern**
  
## Технологии
- Spring Boot – backend framework
- Spring Data JPA – data persistence
- Hibernate – ORM
- Microsoft SQL Server – database
- Maven – build tool
